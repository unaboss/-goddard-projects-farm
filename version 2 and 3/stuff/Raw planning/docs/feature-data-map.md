# Feature → Data Map

| Feature | Data Needed | Source | Format |
|---|---|---|---|
| **Homepage** | Hero image | Admin upload | Image (JPG/PNG/WebP, max ~2MB) |
| | Tagline text | Admin input | Plain text (max 150 characters) |
| | BBBEE certificate / badge | Admin upload | Image (JPG/PNG) or PDF |
| | Navigation links | Hardcoded + generated from active sections | N/A |
| **Livestock Catalog** | Animal photo | Admin upload | Image (JPG/PNG/WebP) |
| | Animal name / ID | Admin input | Plain text (e.g., "Daisy", "Bull #3") |
| | Price range | Admin input | Text or two numbers (min, max in ZAR) |
| | Availability status | Admin toggle | Boolean / enum: "Available" or "Out of Stock" |
| | Pre-filled WhatsApp message template | Generated by system | Text: "Hi, I'm interested in {name} — {price range}. Is it still available?" |
| **Produce Showcase** | Crop photo(s) | Admin upload | Images (JPG/PNG/WebP, multiple per crop) |
| | Crop name | Admin input | Plain text |
| | Crop description | Admin input | Rich text (max ~500 words) |
| | Harvest season months | Admin input | Month ranges (e.g., "March – July") |
| | Typical pricing | Admin input | Text (e.g., "R15 – R25 per kg") |
| | Minimum order quantity | Admin input | Text (e.g., "50kg minimum") |
| | Seasonal badge status | Admin toggle | Enum: "In Season Now" or "Coming Soon" |
| | WhatsApp message template | Generated by system | Text: "Hi, I'm interested in ordering {crop name}." |
| **Crop Voting System** | Voting round season label | Admin input | Text (e.g., "Winter 2026") |
| | Viable crops list | Admin input | List of crop names with short descriptions |
| | Voting deadline | Admin input | Date |
| | Subscriber verification | Mailchimp API | API call — check email exists in audience |
| | Votes cast | User input (checkbox selection) | Multiple crop IDs + voter email, stored locally |
| | Active/inactive round status | System logic | Boolean — based on deadline vs current date |
| | Pop-up email capture | User input | Email address string, validated format |
| | "Thank you" confirmation | Generated by system | Static message text |
| **Contact & Newsletter** | Contact form: name | User input | Text (required) |
| | Contact form: email | User input | Email string (required, validated) |
| | Contact form: phone | User input | Phone string (optional, SA format) |
| | Contact form: message | User input | Text area (required) |
| | Newsletter opt-in checkbox | User input | Boolean (default: unchecked) |
| | Form submission (all fields) | Saved locally | Database record with timestamp |
| | Newsletter subscription | Mailchimp API | API call — add/update subscriber in audience |
| | WhatsApp floating button | Hardcoded | WhatsApp URL with farm number |
| **Admin Panel** | Admin login credentials | Hardcoded / env config | Username + hashed password |
| | All livestock records | Admin CRUD operations | Database rows |
| | All produce records | Admin CRUD operations | Database rows |
| | All voting rounds + results | Admin view / create | Database rows; results are aggregated counts |
| | Hero image & tagline | Admin update | Same format as Homepage data |
